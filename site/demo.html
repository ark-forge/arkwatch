Add Your First Monitor

Configure a monitor for your website. We'll check it every 60 seconds.

# Create your first monitor
$ curl -X POST https://watch.arkforge.fr/api/monitors \
  -H "Authorization: Bearer ark_live_***" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://yourwebsite.com",
    "name": "Production Site",
    "check_interval": 60,
    "alert_email": "your@email.com"
  }'
{
  "status": "success",
  "monitor_id": "mon_xyz789",
  "url": "https://yourwebsite.com",
  "status": "active",
  "next_check": "2026-02-09 17:31:00 UTC"
}
