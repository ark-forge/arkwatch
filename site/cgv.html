1. Objet

Les présentes conditions générales de vente (CGV) régissent l'utilisation des services proposés par ArkForge, accessibles via le site arkforge.fr.

Les services couverts par les présentes CGV sont :

  • ArkWatch : service de monitoring web en temps réel (disponibilité, performances, certificats SSL)

Toute utilisation des services implique l'acceptation pleine et entière des présentes CGV.

3. Description des services

3.1 ArkWatch

ArkWatch est un service de monitoring web qui permet de :

  • Surveiller la disponibilité de sites web et d'APIs
  • Mesurer les temps de réponse
  • Vérifier la validité des certificats SSL
  • Recevoir des alertes en cas d'incident

3.2 MCP EU AI Act

MCP EU AI Act est un serveur MCP (Model Context Protocol) qui permet de :

  • Analyser la classification de risque des systèmes d'IA
  • Vérifier les obligations de conformité selon le Règlement (UE) 2024/1689
  • Générer des rapports de conformité
